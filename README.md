# VTT_TRACKING

This repository mainly use Image-Text-Embedding method and Open re-id.

# Prerequisites

- NVIDIA GPU + CUDA + CuDNN
- Matconvnet + Matlab 2017x
- Pytorch 0.4 + Python 3.6
